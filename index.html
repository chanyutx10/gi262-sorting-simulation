Sorting Algorithm Simulator

800ms

How to Use:

  1. Select Algorithm: Choose Bubble Sort, Insertion Sort, or Selection Sort from the dropdown
  2. Prepare Data: Click "Generate Random Array" for random numbers or "Load Sample Data" for algorithm-specific examples
  3. Start Sorting: Click "Start Sort" to begin the visualization
  4. Step Through: Use the "Step" button to advance one step at a time and see detailed explanations
  5. Control Speed: Adjust the speed slider to make animations faster or slower
  6. Reset: Click "Reset" to start over with the same data or select a different algorithm

Watch the circles change color and move to understand how each algorithm works!
